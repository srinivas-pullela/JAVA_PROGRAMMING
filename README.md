# 📋 Prerequisites

Before running the programs, ensure you have:

* Java JDK 8 or above
* Any Java IDE (VS Code, IntelliJ IDEA, Eclipse)
* Terminal or Command Prompt
* Basic knowledge of Java syntax

Verify Java installation:

```bash
java -version
```

Verify Java Compiler:

```bash
javac -version
```

---

# ▶️ How to Run Java Programs

### Step 1

Clone the repository

```bash
git clone https://github.com/your-username/Java-Programming-Practice.git
```

### Step 2

Navigate into the repository

```bash
cd Java-Programming-Practice
```

### Step 3

Compile the Java file

```bash
javac FileName.java
```

### Step 4

Run the program

```bash
java FileName
```
